-w

-module
--module		Suppress warning messages
-<no->optControl Sisal backend optimization (on by default)
-<no->concurBuild for serial or concurrent (default)
-<no->vectorGenerate code for vectorized loops (off by default)
-<no->aggvectorApply aggressive vectorization (off by default)
-<no->descriptor-bindAssume the shape of arrays through the FORTRAN and C FLI is fixed (default)
-<no->boundsControl bounds checking (on by default)
-<no->slice-throttleControl slice throttle feature that runs short parallel loops serially (on by default)
-keep
--keep-intermediate		Do not discard temporary files
--<no->using-FORTRANUse if linking .o files from FORTRAN
-p <number>
--processors <number>		Partition for this many cpus 1
-iter <number>
--assumed-iterations <number>		Assume this many iterations for loops to drive partition 1000
--parallel-cost-threshold <number>Parallelize loops that have a predicted cost higher than this 7000

-o <string>Executable (default s.out) or object output file name
-main <string>Name of main function (default main)
-tmpdir <string>Where to put temporary files

-externC=<string>

-U*<string>These are passed unmodified to the C and F77 compilers
